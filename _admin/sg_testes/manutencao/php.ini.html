O arquivo PHP.ini

Há duas opções para configurar o PHP na máquina que hospedará a aplicação: a primeira consiste em descompactar o PHP na pasta C:\php\ e substituir o arquivo php.ini por este arquivo com configurações padrão ou configurar o PHP à mão seguindo as descrições abaixo

Guia básico para configuração do PHP.ini para o SiGPOD

1. Renomear o arquivo php.ini-production para php.ini

Alterar memory_limit para um valor maior que 64M de modo que o sigpod disponha de mais memória para executar

Alterar post_max_size para um valor suficientemente grande de maneira que todos os arquivos necessários possam ser enviados para o sigpod (padrão sigpod: >= 256M).

Aletrar upload_max_filesize para um valor compativel com a variável post_max_size

Descomentar as linhas dasextensões utilizadas pelo sigpod: extension=php_ldap.dll, extension=php_curl.dll, extension=php_gd2.dll, extension=php_mysql.dll,

Alterar as configurações regionais para São Paulodate.timezone = America/Sao_Paulo
